## 🎮 Overview

A 3D racing game built in Unity as part of a game jam, featuring physics-based driving, AI-controlled opponents, and a complete playable experience.

What started as a simple scene evolved into a fully functional racing game through iterative development, debugging, and system integration.

---

## 🚗 Features

✨ **Core Gameplay**
- Smooth Rigidbody-based car movement  
- Realistic acceleration and turning  
- Stable physics with downforce and friction  

🤖 **AI System**
- Waypoint-based AI navigation  
- Multiple AI competitors  
- Balanced speed and turning  

🎬 **Game Flow**
- Intro video before gameplay  
- Seamless transition into race  
- Complete playable loop  

🔊 **Audio**
- Engine sound on movement  
- Dynamic pitch based on speed
